# How to Create Vue Project with Vue-cli

#### Install Vue-cli
```
npm install -g vue-cli
```
#### Init Project with Webpack
```
vue init webpack <project-name>
```
####  How to Run Project
**Step1:** 
```
cd <project-name>
```
**Step2:**
```
npm install
```

**Step3:**
```
npm run dev 
```

#### How to Test Project
```
localhost:8080
```
